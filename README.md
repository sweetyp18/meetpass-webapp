MeetPass is a web application designed to streamline student-staff meetings using a token-based approval system.

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: SQL

## Features
- Students can request meetings with staff.
- Staff can approve or decline meeting requests.
- Token-based system for offline meeting approvals.
- Real-time reminders and basic analytics.

## Installation

### Backend
```bash
cd meetpass-backend
npm install
npm start
```
### Frontend
```bash
cd meetpass-frontend
npm install
npm start
```
